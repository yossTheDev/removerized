## 2025-05-14 - Standardizing Shortcut Hints
**Learning:** Tooltip hints for keyboard shortcuts significantly improve user onboarding for power features without cluttering the UI. Standardizing the format to `(Ctrl + Key)` or `(Ctrl + Enter)` ensures consistency and accessibility for keyboard-driven workflows.
**Action:** Consistently use the parenthetical shortcut hint pattern in `TooltipContent` for all interactive editor actions to enhance discoverability.

## 2025-05-14 - Clean Repository Submission
**Learning:** In projects using Yarn Berry/PnP or specific build tools, environment-specific artifacts like `.pnp.cjs`, `tsconfig.tsbuildinfo`, and `dev_server.log` can easily be accidentally staged.
**Action:** Always perform a final manual check and cleanup of autogenerated files before submission to maintain repository hygiene, especially when environment instructions conflict with local setup.

## 2025-03-24 - Consistent Shortcut Affordances
**Learning:** Providing explicit keyboard shortcuts for core utility actions (like Download) significantly reduces friction for power users. Standardizing the `(Ctrl + Key)` hint in tooltips across all toolbar actions makes the UX feel more mature and professional.
**Action:** When adding toolbar buttons for common OS-level actions (Save, Print, Download), always pair them with standard keyboard shortcuts and document them in the UI tooltips.
